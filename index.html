Comandos de Git

  • Git Init: Inicializa un nuevo repositorio Git local
  • Git branch nombre: Crea una nueva rama con el nombre que pongas
  • Git branch: Sirve para ver las ramas y en cual te encuentras actualmente
  • Git checkout nombrDeLaRama: Cambia la rama en que te encuentras
  • Git merge nombreDeLaRama: Con esto uno las ramas
  • Git remote -v: Con esto veo el repositorio remoto
  • Git log: Con esto veo los commits que se hicieron (hisotrial)
  • Git pull origin master: Con esto descargo los cambios del repositorio remoto
  • git push origin main: Con esto subo los cambios de local al repositorio

    • Nota:Al hacer un git commit -m se debe responder ¿Qué hice? y ¿Por qué lo hice?

    Las fases de Git son: Working Directory, Staging Area y Repository
